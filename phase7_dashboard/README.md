# Phase 7: Streamlit Dashboard

This phase consists of a Streamlit app that visualises the insights generated in Phase 5.

## Requirements
- Read structured outputs (`data/weekly_pulse.json` and `data/reviews_classified.json`).
- Ensure no raw text or PII is displayed.
- Display a warning if data is absent.
- No backend required.

## Usage
Run the dashboard locally:
```bash
streamlit run phase7_dashboard/app.py
```
